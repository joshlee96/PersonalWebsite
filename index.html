Projects

Easy Manage

Easy Manage is an Android task managing application which introduces the concept of "task groups". This makes it simple for users to group their tasks under a single category. Easy Manage also allows users to create task groups with their friends! It uses Realm to store persistent client-side data and leverages a Node.js and MongoDB backend.

Portfolio Publisher

Portfolio Publisher is a JavaScript web app which leverages Facebook, LinkedIn, and Github APIs to produce software portfolios.

UWaterloo Reddit Bot

UW Reddit Bot was developed during EngHack 2016 at the University of Waterloo. The bot leverages PRAW (Python Reddit API Wrapper) to retrieve data from the UWaterloo subreddit. When certain keywords are detected, the bot will automatically respond with a formatted message relating to the topic.

Marshmallo

Marshmallo is an Android application designed to store notes and memos. It utilizes Android's SQLite database implementation to store persistent data. The application has create, read, update and delete (CRUD) capabilities. Marshmallo follows Google's material design guidelines.
